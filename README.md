# 🔐 Password Generator – Secure Password Creator

A simple and interactive **Password Generator Web App** built using **HTML, CSS, and JavaScript**.
It allows users to generate strong and secure passwords based on selected criteria.

---

## ✨ Features

* 🔤 Include Uppercase Letters
* 🔡 Include Lowercase Letters
* 🔢 Include Numbers
* 🔣 Include Symbols
* 📏 Adjustable Password Length
* 📋 Copy Password to Clipboard
* ⚡ Instant Password Generation

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## ⚙️ How It Works

* User selects password criteria (uppercase, lowercase, numbers, symbols)
* Password length is chosen using a slider
* JavaScript generates a random password based on selected options
* Copy button allows quick copying to clipboard
